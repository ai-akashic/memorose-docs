# Java

当前 `v1` 运行时（runtime）没有在此发布稳定的官方 Java SDK。

请使用你现有的 HTTP 客户端栈，直接调用文档中记录的 REST 路由。

典型流程：

1. 通过 `/v1/dashboard/auth/login` 登录
2. 保存 Bearer 令牌（token）
3. 调用 `/v1/users/:user_id/streams/:stream_id/events`
4. 调用 `/v1/users/:user_id/streams/:stream_id/retrieve`
