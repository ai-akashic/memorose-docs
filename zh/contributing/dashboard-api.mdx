# Dashboard API

仪表盘（Dashboard）路由嵌套在 `/v1/dashboard` 下，为运维 UI 提供支持。这些是内部端点（endpoint）——终端用户直接与仪表盘 UI 交互。

## 认证

```http
POST /v1/dashboard/auth/login
```

默认凭据为 `admin` / `admin`，除非你设置了 `DASHBOARD_ADMIN_PASSWORD`。

## 路由

### 认证

| 方法 | 端点 | 用途 |
| --- | --- | --- |
| `POST` | `/v1/dashboard/auth/login` | 获取仪表盘 JWT 令牌（token） |
| `POST` | `/v1/dashboard/auth/password` | 修改仪表盘密码 |

### 统计与记忆

| 方法 | 端点 | 用途 |
| --- | --- | --- |
| `GET` | `/v1/dashboard/stats` | 全局系统统计 |
| `GET` | `/v1/dashboard/memories` | 列出记忆单元（memory units） |
| `GET` | `/v1/dashboard/memories/:id` | 获取特定记忆的详情 |

### 图谱与搜索

| 方法 | 端点 | 用途 |
| --- | --- | --- |
| `GET` | `/v1/dashboard/graph` | 获取图谱数据用于探索 |
| `POST` | `/v1/dashboard/search` | 搜索试验场端点 |
| `POST` | `/v1/dashboard/chat` | 聊天/查询试验场 |

### 遗忘与纠正

| 方法 | 端点 | 用途 |
| --- | --- | --- |
| `POST` | `/v1/dashboard/forget/preview` | 预览语义遗忘（semantic forgetting）计划 |
| `POST` | `/v1/dashboard/forget/execute` | 执行遗忘计划 |
| `POST` | `/v1/dashboard/corrections/semantic/preview` | 预览语义纠正 |
| `POST` | `/v1/dashboard/corrections/semantic/execute` | 执行语义纠正 |
| `POST` | `/v1/dashboard/corrections/manual` | 手动纠正 |
| `GET` | `/v1/dashboard/corrections/reviews` | 获取审核队列 |

### 运维操作

| 方法 | 端点 | 用途 |
| --- | --- | --- |
| `GET` | `/v1/dashboard/config` | 查看运行时配置 |
| `GET` | `/v1/dashboard/agents` | 监控代理（agent）指标 |
| `GET` | `/v1/dashboard/cluster/status` | 检查集群健康状态和节点 |
| `GET` | `/v1/dashboard/organizations` | 列出托管的组织 |
| `POST` | `/v1/dashboard/organizations` | 创建组织 |
